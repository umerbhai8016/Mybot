🤖 About

CodeXGPT ek professional AI chatbot hai. Yeh index.html ek fully self-contained single file hai — koi server, npm, ya setup nahi chahiye. Seedha browser mein kholo!

✨ Features

  • 🔐 User Authentication — Signup & Login with password hashing
  • 💬 Multiple Chat Sessions — Create, switch, delete anytime
  • 🤖 AI Responses — Powered by Claude (Anthropic API)
  • 🖼 Image Upload — Upload or paste images for vision analysis
  • 🎤 Voice Input — Microphone se bolo
  • ✏ Auto Chat Titles — 2nd message ke baad AI title generate karta hai
  • 💾 Persistent Storage — Sab data localStorage mein save hota hai<
